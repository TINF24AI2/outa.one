# App Architechture 

![Image showcasing the app architechture.](/docs/assets/architecture.svg)

## Client  
The client is the interface for the end user. Software keys are provided by administrators and downloaded by users here.

## Frontend  
The frontend provides the visual component for the client.

## Backend  
The backend represents the underlying system of the website. It handles logic and control, operating outside the user’s view.

## Database  
The database stores application-related data, such as user accounts and software keys.

## Mail Server (external)  
An external mail server ensures reliable email delivery. This improves performance and helps guarantee that emails reach users.
